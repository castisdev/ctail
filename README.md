# CTail

`ctail` is a simple python script to tail log files(like tail -f).

##Installation

To install, simply:

```
$ ./install.sh
```

## Features

- Countinuously open and tail latest file in the directory.
- Highlight columns
- Highlight Error, Fail, Warning level keywords
- Highlight values in brackets.
