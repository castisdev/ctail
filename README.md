# CTail

최신 파일을 계속해서 tail -f 해주는 python(>=2.4) 스크립트

## 기능

- 폴더 내 최신 파일을 지속적으로 열어서 tail -f 해줍니다.
- cilog 포멧에 대한 하이라이트
- 괄호 내 단어들에 대한 하이라이트

## 설치

```
$ sudo ./install.sh
```
